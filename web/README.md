# daily-news-dashboard

Astro 5 + Tailwind 4 + TypeScript static site for the daily-news investment dashboard.

## Build prerequisite

Node 22+. Install via `nvm install 22 && nvm use 22`.

(Confirmed working on Node 25.6.1 too.)

## Dev / build

```bash
npm install
npm run dev          # http://localhost:4321
npm run build        # → dist/
npm run preview      # serve dist/ locally
```

`npm run build` runs `astro check` (type-checking) before `astro build`.

## Pages

| Route | Source | Data file |
|---|---|---|
| `/` | `src/pages/index.astro` | `src/data/overview.json`, `src/data/screens.json`, `src/data/decisions.json` |
| `/stocks/[ticker]` | `src/pages/stocks/[ticker].astro` | `src/data/stocks/<TICKER>.json` |

The homepage is an investor research queue: high-priority research, validation
queue, risk radar, watchlist decision table, market signals, and source
evidence. It intentionally avoids a raw news dump as the first screen.

## Data refresh

JSON under `src/data/` is regenerated by the Python pipeline:

```bash
cd ..
python dashboard_export.py --db data/news.db --output web/src/data
```

Or via the launchd wrapper:

```bash
cd ..
bash launchd/export-dashboard-data.sh
```

## Deployment

See [`../docs/dashboard-deployment.md`](../docs/dashboard-deployment.md).

The live site is served by a local `python3 -m http.server` on
`127.0.0.1:8055`, proxied through Cloudflare Tunnel to
`https://invest.aihost.dev/`. It is not a Cloudflare Pages deployment.
